# 📈 Sales Prediction

## Overview
A machine learning project that predicts product sales based 
on advertising spend across TV, Radio, and Newspaper platforms 
using Linear Regression.

## Dataset
- Source: CodeAlpha (Advertising CSV)
- 200 samples
- Features: TV, Radio, Newspaper (ad spend), Sales

## Model
- Algorithm: Linear Regression
- Test Size: 20%
- Evaluation: R² Score & RMSE

## Results
- TV and Radio advertising had the highest impact on sales
- Newspaper advertising had minimal effect on sales outcomes
- Model successfully forecasts sales from advertising budget

## Key Insights
- Increasing TV ad spend has the strongest positive effect on sales
- Radio ads are more cost-effective per unit of sales generated
- Businesses should prioritize TV and Radio over Newspaper ads

## Tools & Libraries
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## How to Run
1. Download `Advertising.csv` from the repository
